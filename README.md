# Castrol Supply Chain Decision Support System

## Overview

A modular supply chain analytics and decision support system developed for the Castrol Case Competition.

## Features

- Demand Analytics
- Inventory Planning
- Production Optimization
- Plant Capacity Analysis
- Network Analysis
- Cost Analysis
- Executive Dashboard

## Tech Stack

- Python
- Pandas
- Plotly
- Streamlit
- OpenPyXL

## Project Structure

```text
src/
├── analyzer
├── dashboard
├── data
├── engines
└── optimizer
```

## Run

```bash
python main.py
```

Launch dashboard

```bash
streamlit run src/dashboard/app.py
```

## Dashboard

Interactive Streamlit dashboard with:

- Executive KPIs
- Demand Analytics
- Inventory Analytics
- Plant Analytics
- Interactive Filters
- Download Reports
